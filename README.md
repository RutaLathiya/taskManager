# 🚀 Task Manager Web App

## 📌 What I Built

I built a simple Task Manager web application using Python, Flask, HTML, and SQLite.

The application allows users to:

* ➕ Add tasks
* ✏️ Edit tasks
* ❌ Delete tasks
* ✅ Mark tasks as completed
* 🔥 Set task priority levels

The tasks are displayed dynamically in a table format using Flask templates.

---

## 💾 Storage Approach Used and Why

I used SQLite as the storage solution because it is lightweight, file-based, and easy to integrate with Flask. It does not require a separate database server, which makes development simple and fast for small applications.

SQLite also provides structured persistent storage and makes CRUD operations like insert, update, delete, and fetch easier compared to storing data in other Options.
